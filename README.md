# 🌍 EarthLens

A 3D interactive globe explorer. Explore Earth in impossible ways.

## Modes
- ⚖️ **Scale** — compare sizes of regions
- 🕳️ **Dig** — dig through Earth to the antipodal point
- 💥 **Blast** — drop impacts and see blast radii
- 🔴 **Latitude** — find all cities on the same latitude
- 🌓 **Day/Night** — real-time day/night terminator

## Setup
```bash
npm install
npm run dev
```

## Deploy (free)
- **Vercel**: `vercel --prod`
- **Netlify**: drag `dist/` folder to netlify.com
- **GitHub Pages**: push `dist/` to `gh-pages` branch

## Tech Stack
- React + Vite + TypeScript
- globe.gl (Three.js)
- Framer Motion
- Zero backend, zero cost
